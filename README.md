# Fleet Failure AI Agent

Machine learning + AI agent system for analyzing fleet machine failures.

## Overview
This project builds a predictive maintenance pipeline using machine learning models and an AI agent that allows fleet managers to ask natural language questions about model performance and machine failure drivers.

## Models
- Logistic Regression
- XGBoost

## Features
- Exploratory Data Analysis (EDA)
- Model performance comparison
- Feature importance analysis
- AI-powered fleet assistant

## Example Questions
- "Is the dataset imbalanced?"
- "Which model performs better?"
- "What features drive machine failure?"
- "What should mechanics monitor?"

## Tech Stack
Python, Scikit-learn, XGBoost, LangChain, LangGraph, Pandas
